# CNN-C

## Introduce

This is a simple cnn program with C.

## Build

Build it with make, and just test by default.
> make CC=gcc \
> make CC=gcc run \

Build it with only training:
> make CC=gcc TRAIN=1 \

Build it with training and test:
> make CC=gcc TRAIN=1 TEST=1
